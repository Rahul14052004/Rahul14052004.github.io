QUADRA : A Counterspeech Generation Model

Co-developed QUADRA, an intent-conditioned counterspeech generation framework leveraging HateBERT and BART for controlled response generation. Implemented fusion techniques (Linear, Cross-Attention, Shared Fusion) to integrate hate speech and intent embeddings, and introduced DialoRank for zero-shot evaluation. QUADRA achieved superior performance compared to DialoGPT and GPS baselines across multiple NLP metrics.

Technologies & Skills: Large Language Models (LLM), Fine-Tuning, Hugging Face, NLP

SmartReachAI: An Email Campaign Management System

Developed an AI-driven email automation platform integrating Mistral, Gemma, and a custom Spam Transformer in a RAG framework (PyTorch + Hugging Face) to enable spam-compliant, personalized communications under DPDP Act guidelines. Built a scalable backend pipeline with Send Time Optimization (STO) powered by XGBoost and clustering on PostgreSQL. Implemented Celery for scheduling, Redis for caching, and a modern Next.js frontend for analytics visualization.

Technologies & Skills: Django, Next.js, LLMs, RAG, Agile Project Management, XGBoost, Performance Dashboards

Robo Bartender: An AI-Driven Autonomous Bartending System

An intelligent robotic bartender system that combines advanced computer vision, natural language processing, and precise robotic manipulation to create cocktails through voice commands. The system integrates real-time speech recognition, AI-powered conversation, color-based bottle detection, and robotic arm control to deliver a seamless bartending experience.

Technologies & Skills: LLMs, Groq Inference, ROS, Cobot Arms

Probabilistic Pundit: A Multi-Agent LLM Framework for Dream Team Creation in Fantasy Football

Probabilistic Pundit is an AI-powered system that generates optimized Fantasy Premier League (FPL) squads using a combination of deterministic ranking algorithms and multi-agent Large Language Model reasoning. The project integrates a FastAPI backend, Streamlit frontend, and a production-oriented MLOps pipeline.

Technologies & Skills: LLMs, Cloud-Based Inference, Streamlit, FastAPI
