Simulating Premier League Standings with Machine Learning

Overview

Premier Insights is a machine learning project aimed at predicting the outcomes of English Premier League (EPL) matches and simulating final league standings. By integrating models such as XGBoost, Random Forest, and Gradient Boosting with Elo ratings and xG statistics, the project enhances the accuracy of match predictions.

Technologies Used

  • Python
  • XGBoost, Random Forest, Gradient Boosting
  • BeautifulSoup (for web scraping player data)
  • Kaggle Datasets (historical match data)

Model Performance

Model Accuracy
Logistic Regression 54.34%
Support Vector Machine (SVM) 56.97%
Decision Trees 61.57%
Artificial Neural Network 66.84%
Multi-Layer Perceptron 69.73%
Random Forest 73.55%
xGBoost 74.07%
Custom Ensemble Model 75.52%
Gradient Boosting 76.84%

Class-Wise Performance

Gradient Boosting and xGBoost models performed exceptionally well at predicting home wins and draws, showcasing their effectiveness for league simulations.
