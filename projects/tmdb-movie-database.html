A movie search portal integrated with TMDB API and streaming platform availability.

Overview

MovieScore is a movie search portal that allows users to search for movie descriptions, cast details, and availability across major streaming platforms. The application connects to TMDB (The Movie Database) API and cross-references data with Netflix, Hulu, Amazon Prime, and Disney+ to provide streaming availability.

Motivation

With the rise of streaming services, users often need to search across multiple platforms to find where a movie is available. MovieScore aims to simplify this by offering a unified portal that retrieves movie details and lists streaming availability, streamlining the user experience.

User Interface (GUI)

The GUI was built using Tkinter, allowing users to input movie titles and receive instant feedback. A simple but effective layout displays movie details alongside platform availability, enhancing usability and user experience.

Testing Approach

Testing involved verifying individual functions and components:

  • A rough sketch of the GUI was translated into Tkinter.
  • Specific movies were tested using the TMDB API to verify accuracy.
  • A CSV file containing movies and platform data was cross-referenced to validate streaming availability.
  • Each API call was monitored and debugged individually to ensure correct data retrieval.

Challenges and Learning

One of the primary challenges was managing API rate limits and handling platform discrepancies between regions. Additionally, synchronizing TMDB data with CSV records required careful filtering and validation. This project improved our understanding of working with APIs, handling GUI events, and managing external data sources.
