Automating music classification using machine learning and deep learning techniques.

Overview

This project focuses on developing a machine learning model to classify music into genres using audio features and spectrogram analysis. The goal is to assist with playlist generation, content organization, and music analysis for industry applications.

Motivation

With the rise of streaming platforms and vast music libraries, automatic music classification plays a crucial role in improving user experience by categorizing songs efficiently. This classifier could streamline radio stations, aid in music licensing, and provide insights for music producers and DJs.

Dataset

We used the GTZAN Genre Collection, a popular dataset comprising 10 genres (rock, classical, jazz, etc.), with 100 audio samples per genre. Each sample is 30 seconds long.

Technologies Used

  • Python
  • TensorFlow/Keras
  • Scikit-learn
  • Librosa (audio processing)
  • Mel-Spectrograms and MFCCs

Models Implemented

  • K-Nearest Neighbors (KNN): Simple and intuitive, but limited accuracy (67%).
  • Logistic Regression: Provides baseline accuracy of 66%.
  • Decision Trees: Accuracy of 53%, prone to overfitting.
  • Convolutional Neural Networks (CNN): Achieved 86% accuracy by processing mel-spectrograms.
